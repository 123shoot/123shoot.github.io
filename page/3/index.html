发表于 2018-11-18 | 分类于 日记

今天是11月18日，星期日，小雨

首先昨天没有写日记的原因是因为忘了。昨天因为新买了饥荒哈姆雷特的DLC，所以就玩了一会儿，结果蚁后实在是太太太太太太太太太太太难了，所以我回档n次，也搞到了半夜两点多，接着就上床睡觉。自然也是忘了写日记了。

第二个事情是我现在是洗完澡躺在床上，一边写日记，一边左上角小窗播放着hithere的街坊，看看外国人对某些问题的看法挺好玩的。而且这节目里问题贼犀利，比如什么美国人怎么看待印第安人问题，美国人怎么看待美国“世界警察”的问题等等。

第三件事情是要回顾下今天的学习内容和一天的时间耗费。IC测试的最后一节课周多老师是可以玩，可以学习，但是一定不要每天浑浑噩噩，稀里糊涂地过完一天。我感觉我就是稀里糊涂过完了大学三年。。。言归正传，由于昨天晚上睡的太晚，今天上午基本废了，吃过中饭后，两点钟左右去教室学习，然后晚上九点多回寝室，感觉也没有学到啥东西。明天要考IC测试，今天肯定是学习相关的内容了。

我现在脑海里的整体框架就是IC测试第一章讲的是测试的基本定义和种类，然后是比较重要的故障覆盖率，良率和DPM的关系，有一个图像，不知道会不会考。第二章讲的是故障的种类比如桥接故障什么的，但是重点是故障表的使用以及故障的等效、支配进而进行化简。那不可能任何故障都用故障表，所以第三章讲的是给你一个测试矢量，可以测试哪些故障？常用的方法有并行测试法，也就是计算机多少多少位那个，但是这个只能检测对应的几种故障能不能测试得到，并不实用，所以就有了演绎测试法，就是利用一些逻辑关系，不同门的控制端不同等等，从输入端向着输出端一点点的推进，最后得出能够测试出的错误。演绎法变形就得到了并发故障模拟法，其实就是把研究连线换成了研究门，还麻烦一点，但是适用于测试信号经常发生变化，只要改动一小部分内容就行了。

第四章讲的是怀疑一个逻辑电路有一个或几个什么故障，那么怎么产生测试信号去测试？常用的是异或法、布尔差分法和D算法，其中D算法的内容是很复杂的，比较重点的概念有 奇异立方、D立方、原始D立方、传播D立方、驱赶D立方等。

第五章讲的是CC，SC，CO，SO，就是控制度和观测度，我感觉有点像二叉树的深度的概念。这一部分内容还不是很难。

第六章第七章还没有看，应该会考的非常少或者不考，睡前再看一下吧。

下午看了会周多的时候有点无聊，就复习下链表，我想写个那种提示你输入学生的名字学号姓名等等，然后只要输入exit就自动结束，写了一大会儿也没能实现。我思考了好一会儿发现一个问题，书上创建链表的时候都是先建立一个结点，然后才是while循环，循环里再判断是不是第一个结点，我原来想这样不是多此一举吗？我写的时候都是直接放在while里创建结点，也就是一开始并不创建一个结点，然后就有问题了，就是我没法判断输入的exit，因为判断条件是写在while里的，但是我的一开始的指针都是野指针，那比如pnew->sName，虽然不报错，但是一运行就意外退出，因为这时候pnew是野指针，又怎么可能存在sName这个元素呢？可见我的水平还是。。唉。

不知不觉写日记写了半个多小时了，哦对了还有最后一件事，就是昨天开了美团的外卖会员，因为周年庆还是什么，只要三块钱，送三十块的红包哦，我还有饿了么的会员，另外因为我是双卡双待饿了么和美团都是有两个号哈哈，最近吃外卖感觉好便宜，一般都是不到十块钱。

就这么多了，都快十二点了，我再看会儿周多小姐姐的PPT和题目，将近一点再睡吧，嗯就这样，晚安～

雨声与昏黄的灯光

发表于 2018-11-12 | 分类于 杂谈

今天下午本来打算去图书馆学习，把刘伟景的论文给解决掉，然而一点多到了图书馆，直到现在我还没有打开书本。原因是什么呢？就是因为我打开了更新之后的网易云音乐，然后发现多了一个背景音的功能，听了一会儿后就无法自拔，实在是太有感觉了。

最有感觉的就是“雷声”这个背景音，天哪，实在是太真实了，一戴上耳机沉浸感就非常强，思绪瞬间就回到了高三的雷雨交加的夜晚。

下午下的是暴雨，到了晚上，雨虽然小了一点儿，但是雷声确是此起彼伏。九点多下了晚自习，我骑着单车匆匆忙忙地赶回家里，脱掉雨衣，身上有一半的地方都淋湿了，我妈微笑着递给我一条毛巾：“淋坏了吧？”，我拿着毛巾擦一擦湿漉漉的头发，来到我的房间：房门是一直开着的，里面的白炽灯已经打开了，泛着昏黄的灯光。我打开纱门，吱呀一声，片刻之后，“嘭”地一下关住。书桌是老式的长条桌，上面的红漆已经变得斑驳，桌板的缝隙里是灰尘和纸屑，桌子上还有蓝色的，紫色的涂鸦线条，和深深浅浅的圆珠笔戳的点点。我打开书包的拉链，一本本书的书角已经湿了，不过并无大碍。

过了一会儿，妈妈走进来，递给我一杯温热的牛奶，或是鸡蛋打到热水里搅拌而成的鸡蛋茶。我还清楚的记得用的杯子是大大的玻璃杯，喝啤酒用的那种。接着，妈妈会把暖瓶拿进来，如果我的洗脚盆的水还没有倒掉，她就会一边轻声地唠叨，一边帮我端出去倒掉。差不多十一点，爸爸妈妈都睡了，只有我还在做作业。作业做完之后，睡前洗脚的时候，有时我会拿出借同学的意林或是其他什么杂志，翻翻看看，毕竟高三的娱乐互动实在是不多。

当我从回忆中剥离出来之后，我才发现已经过了一个多小时，窗外只是阴天，我身处图书馆，这里是明亮的LED灯光。可是我实在是太喜欢回忆里的场景了，仔细地回忆，仿佛品茶一般，又感动地想落泪。

我不知道像我这样喜欢回忆往事，而后又有诸多感慨，心生唏嘘的人多不多；但是我觉得如果有这样的一家店，它能模拟回忆里的这种场景，比如房间外是雷雨声，房间内是昏黄的灯光，有着一段历史的老桌子，桌子上泛黄的书页，右手边是热气腾腾的一杯热茶，暂时放下心里所有的杂念，纷扰，只是静静地发呆，或是看书，或是回忆往事，都再美好不过了。

如果真的有这样的店，其实要做的也并不多。一个音效很好的环绕声的小房间，一些旧物也就够了。

再说点玩笑话，就是模拟这样的一个场景，都有很多发挥的空间。比如说房间的格调。有的人童年里的房间的印象可能是洁白的墙壁，光滑的大理石地板，明亮的窗户；但是有的人记忆里的房间是斑驳的墙面，黑乎乎的水泥地，一张小木床，下雨的时候，还会滴答滴答地漏水。而对于我而言，我童年里的房间是后者。另外，我恰恰觉得这种略显破旧的房间，在外面的雷声雨声的衬托下，反而更显的温暖，充满回忆。

（完）
